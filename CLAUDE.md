# Jorge Freelancer – Marketing-Seite

## Projekt
Einzelne `index.html` – Freelancer-Marketing-Seite (Deutsch, Immobilien & Digitale Dienstleistungen).

## Remote
- Repository: https://github.com/jorgepnt-design/jorge-freelancer
- Branch: `main`

## Workflow
Nach **jeder Änderung** an `index.html`:
1. `git add index.html`
2. `git commit -m "<kurze Beschreibung der Änderung>"`
3. `git push origin main`

Push immer direkt ausführen – keine Bestätigung abwarten.

## Berechtigungen
Alle Datei- und Git-Operationen sind vorab genehmigt. Kein Nachfragen nötig.
